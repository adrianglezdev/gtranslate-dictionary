Features

Everything you need.
Nothing you don't.

🗂️
Per-language dictionary
Configure independent overrides for each target language. English, German, French — each with its own set of rules.
🖱️
Visual interface
Add, edit and delete entries directly in the WordPress admin. No JSON files, no code, no FTP.
🏠
Configurable source language
Your site isn't in Spanish? No problem. Set any language as the origin and the plugin adapts automatically.
⚡
Zero-config frontend
Overrides are injected automatically via a tiny JS snippet. No shortcodes, no template edits required.
🔒
Free forever
No premium plan. No license key. No feature walls. Works entirely with the free tier of GTranslate.
🧠
Smart detection
Detects the active language via cookie, URL parameter or HTML lang attribute — covering all GTranslate modes.

How it works

Up and running in minutes

01
Download
Grab the .zip from GitHub Releases. No account needed.
02
Install
Upload via Plugins → Add New → Upload Plugin in WordPress.
03
Configure
Open GT Dictionary, set your source language, add target languages and enter your overrides.
04
Clear cache
Save and clear your site cache. Overrides appear immediately on the frontend.
